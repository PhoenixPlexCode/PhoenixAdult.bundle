    - `✓ - **Title/Actor only** (Some subtitle or note)`

    - Use the subtitle to denote if a site uses SceneID or Direct URL.
    - `❌ - **SceneID/Direct URL** (Some subtitle or note)`

```
+ #### Some Example Site | {✅, ✓, ❌}

```

```
+ #### Some Example Network | {✅, ✓, ❌}
```
